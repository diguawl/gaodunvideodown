# gaodunvideodown
高顿网校课程下载

所需准备：

一台 Windows 10 或以上系统的电脑

Chrome 或 Edge 浏览器（推荐使用）

网络稳定，并已登录高顿账号

下载辅助工具（获取软件可以关注公众号：地瓜网络技术）


操作步骤

第一步：使用电脑登录高顿网校
请务必使用电脑端浏览器访问 高顿网校官网，并登录你的账号，进入你想下载的课程页面，正常播放视频。

⚠️ 注意：手机端或APP内无法完成后续操作，请使用电脑。

第二步：获取网页 Cookie

按F12，去找到对应的cookie。

第三步：将 Cookie 粘贴到下载工具中

打开你准备好的视频下载工具（支持高顿课程解析的专用工具）。

将刚才复制的 Cookie 粘贴到工具指定的输入框中。

工具会自动识别你的账号信息，并解析出你需要下载的视频。

第四步：选择并开始下载


第五步：查看下载完成的视频
下载完成后，你可以在工具设置的默认保存路径（如 D:\Gaodun_Videos\）中找到所有课程视频。这些视频无加密、无水印，支持在任何播放器中离线观看。
